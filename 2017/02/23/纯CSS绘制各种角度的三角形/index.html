以下是纯CSS绘制各种角度的三角形的方法。

Triangle Up

代码：

#triangle-up{
    width:0;
    height:0;
    border-bottom: 80px solid #00ffff;
    border-left:40px solid transparent;
    border-right:40px solid transparent;
}
